# 08. 우선순위 & 로드맵

## P0 — 출시 필수

- **계정·팀**: 카카오 로그인, 팀 생성·초대(코드 만료 설정), 마이페이지
- **회의 전**: 회의 생성, 아젠다 검토·배포, 알림
- **회의 중**: 회의 룸(다인 접속·미접속 경고), **Web Speech API STT 통합**(브라우저 내장, ~60초 강제 종료 자동 재시작 처리), **안건 추적 ★**, 안건별 요약, **실시간 기여도 바 ★**, 결정·액션 빠른 입력
- **회의 후**: 그루핑 정리, AI 회의 종합 정리, 검토·확정, 다음 회의 아젠다 생성(LLM), 기여도 대시보드(3축 지표+신뢰도, 액션 난이도 가중), 액션 대시보드, 교수 제출용 PDF
- **알림**: 인앱 + 이메일, 회의 5분 전
- **운영**: 랜딩 페이지, 약관

> **알려진 한계 (MVP)**: Web Speech API는 음성을 브라우저 외부(Google) 서버로 전송하므로 데이터 주권·프라이버시 측면의 한계가 있고, 약 60초 강제 종료가 있어 자동 재시작으로 방어한다. 디바이스·환경 편차로 인식 품질이 갈릴 수 있어 신뢰도 라벨([06](06-기여도-산정.md))에 반영한다. 이 한계는 V2의 RealtimeSTT 로컬 추론 전환으로 해소할 계획이다.

## P1 — 출시 후

- 사전 의견 수집 폼
- 회의 2시간 전 준비 체크 푸시, 준비도 현황판
- 마감일 자동 푸시
- 기여도 리포트 내보내기

## V2

- **데스크탑 앱(Electron) + RealtimeSTT 로컬 추론** — 음성을 외부(Google) 서버로 보내지 않는 로컬 STT로 데이터 주권 회복, faster-whisper 기반 (상세 계획: [05](05-STT-음성-처리.md)·[07](07-Electron-구현.md))
- **always-on-top 보조 창** — 데스크탑 전환 시 별도 브라우저 창 대신 네이티브 항상 위 보조 창 제공
- AI 자동 안건 추적
- 종합 점수 표준화 (팀 내 정규화)
- 동료 평가
- Discord/Zoom 봇 직접 연동
- 모바일 대응
- 자동 업데이트 인프라
- 오프라인 대면 회의 지원 (현재는 온라인 회의 보조에 집중)
- 과제 업로드·검토 (클라우드 드라이브 연동)
- AI 챗봇 (회의 상황·일정 확인)
