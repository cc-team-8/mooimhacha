# 무임하차 — 팀플 협업 보조 도구

대학생 팀플의 무임승차 문제를, 회의 중 **실시간 기여도 가시화**로 해결하는 웹앱. (데스크탑 앱은 V2)

## 제품 정의

- **형태**: 회의 앱 옆에 띄우는 보조 도구 — MVP는 **웹앱**(회의 중 보조 창은 별도 브라우저 창). 데스크탑(Electron)·always-on-top은 **V2**
- **회의를 대체하지 않음** — 기존 회의 도구 옆에서 보조
- **온라인 회의 보조에 집중** — 오프라인 대면 회의 지원은 V2
- **타겟**: 대학생 팀플 (팀 인원 상한 6명, 하드 제한)

## 구현 시 반드시 지켜야 할 제약

| 제약               | 내용                                                               |
| ------------------ | ------------------------------------------------------------------ |
| 운영 비용 0원      | MVP STT는 브라우저 내장 Web Speech API. 유료 STT API 금지. 로컬 추론(RealtimeSTT)은 V2 |
| 음성 원본 미저장   | 우리 서버에는 변환된 텍스트만 전송·저장. 단 Web Speech는 음성을 외부(Google)로 보냄(MVP 한계, V2에 해소) |
| 방식 B (각자 녹음) | 각자 본인 마이크로 본인 발화만 수집 → 화자 분리 불필요             |
| HTTPS 필수         | 마이크·Web Speech가 보안 컨텍스트 요구 → 도메인 + TLS(Caddy)        |
| 보조 창            | 회의 중 보조 창은 별도 브라우저 창(폭 400px). always-on-top은 V2   |
| LLM 사용           | GPT-4o-mini. 회의 중 안건별 요약(안건당 1회) + 회의 후 종합 1회 + 다음 회의 안건 생성 1회 |

## 핵심 기능 두 가지 (★)

1. **안건 추적** — 회의 안건의 진행 상태·시간 초과를 실시간 시각화
2. **실시간 기여도 바** — 회의 중에는 발언 비중을 실시간 표시 (역할 수행도 포함 3축 종합은 회의 후 대시보드)
